# Icons

Version: 1.0

Status: Frozen

# Philosophy

Icons support text.

Icons should never replace text where clarity is reduced.

The interface should remain understandable without relying solely on icons.

# Icon Library

Lucide React

No additional icon libraries are permitted.

# Sizes

Small

16px

Default

20px

Large

24px

# Usage

Navigation

Buttons

Status

Tables

Empty States

Notifications

Inputs

# Alignment

Icons align with text.

Spacing between icon and text

8px

# Style

Outlined icons only.

Filled icons are not used.

Maintain consistent stroke width.

# Colors

Icons inherit text color unless indicating status.

Status colors

Success

Warning

Error

Info

Must follow the Color System.

# Interactive Icons

Support

Hover

Focus

Disabled

Keyboard navigation

# Decorative Icons

Decorative icons should be hidden from screen readers.

# Accessibility

Meaningful icons require accessible labels.

Buttons using icons only must include an accessible name.

# Rules

Never mix icon styles.

Do not use oversized icons.

Avoid icons without supporting context.
