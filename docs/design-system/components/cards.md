# Cards

Version: 1.0

Status: Frozen

# Philosophy

Cards group related information.

Cards should improve organization, not create visual clutter.

Avoid nesting cards inside cards.

# Usage

Cards are used for

Dashboard summaries

Product previews

Customer information

Settings sections

Analytics widgets

Empty states

Cards should not be used for long-form content.

# Rules

Do not overload cards.

Prefer multiple focused cards over one large card.

Avoid unnecessary borders.

Maintain consistent spacing.
