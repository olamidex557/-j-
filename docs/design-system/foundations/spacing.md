# Spacing System

Version: 1.0

Status: Frozen

# Philosophy

Spacing creates structure.

Use spacing consistently to establish rhythm throughout the interface.

Never guess spacing values.

Always use the spacing scale.

# Base Unit

4px

All spacing must be a multiple of 4.

Spacing must remain consistent across Dashboard and Storefront.
