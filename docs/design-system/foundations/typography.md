# Typography

Version: 1.0

Status: Frozen

# Philosophy

Typography creates hierarchy.

Use typography to guide attention rather than relying on excessive color or decoration.

The interface should remain highly readable across all screen sizes.

Maintain consistent hierarchy across every page.
