# Oja Design System

Version: 1.0

Status: Frozen

Last Updated: July 2026

# Overview

The Oja Design System defines the visual language used across the platform.

It ensures every interface is

- Consistent
- Accessible
- Responsive
- Scalable

The design system applies to

- Merchant Dashboard
- Customer Storefront
- Future Mobile Application
- Internal Admin Portal

# Design Principles

The interface should feel

Professional

Simple

Fast

Modern

Trustworthy

The interface should never feel

Cluttered

Playful

Over-animated

Distracting

# Design Philosophy

Less is more.

Whitespace improves readability.

Typography creates hierarchy.

Color communicates meaning.

Motion provides feedback.

Consistency builds trust.

# Accessibility

Keyboard navigation required.

Visible focus states required.

Proper color contrast required.

ARIA attributes required where applicable.

Accessibility is mandatory.

# Responsive Design
